## Installation

**Installation via `requirements.txt`**:

```shell
$ git clone https://github.com/Ritu25M/Relief_Camp_Inventory.git
$ cd 
$ python3 -m venv myenv
$ source myenv/bin/activate
$ pip3 install -r requirements.txt
$ flask run
```
